Завдання 2: Бустери та Суперсили ⚡🧪

Твій предмет уже виглядає круто, але тепер час зробити його функціональним. У цьому завданні ми використаємо подію Activated, щоб при кліку мишкою інструмент впливав на параметри персонажа: рятував життя, давав суперстрибок або навіть спрацьовував як хитра пастка.

💡 Вибери функцію для свого інструмента:

Енергетик (JumpPower)
Дає здатність перестрибувати великі перешкоди. Збільшує силу стрибка гравця до 100 або 150.
Аптечка (Health)
Рятує в небезпечних ситуаціях. Відновлює здоров'я гравця до максимуму (humanoid.Health = 100).
Отруйне Зілля (Damage)
Предмет-пастка! Виглядає як звичайний напій, але при використанні віднімає здоров'я або робить його рівним 0.

Алгоритм програмування Tool:

1
Скрипт: Додай звичайний Script всередину свого Tool. Створи змінну для інструмента: local tool = script.Parent.
2
Подія кліку: Використай подію Activated. Вона спрацьовує, коли гравець тримає предмет у руках і натискає ліву кнопку миші. Відкрий функцію Connect(function().
3
Пошук гравця: Всередині функції знайди персонажа, який тримає предмет (tool.Parent), та його Humanoid через команду FindFirstChild.
4
Суперсила: Зміни потрібну властивість Гуманоїда. Наприклад, humanoid.JumpPower = 100 або humanoid.Health = 100. 
local tool = script.Parent

tool.Activated:Connect(function()
    local character = tool.Parent
    local humanoid = character:FindFirstChild("Humanoid")
    
    if humanoid then
        -- Зміни властивість нижче (JumpPower або Health)
        humanoid.JumpPower = 100 
    end
end)
Увага (Часта помилка): Перевіряй регістр літер! Властивості пишуться з великої літери посередині: WalkSpeed, JumpPower, Health. Якщо напишеш з малої, скрипт видасть помилку в Output.

🎯 Перевір себе: Анатомія Гуманоїда

Щоб керувати здібностями гравця, треба чітко знати властивості його Гуманоїда. З'єднай команди та терміни з їхнім правильним значенням.
